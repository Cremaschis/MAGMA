# MAGMA BRAND SYSTEM

Magma is not just a caffeine product.
It is a performance ritual brand.

The website should feel:
- premium
- energetic
- fast
- addictive
- modern
- culturally relevant
- mobile-first
- TikTok-native

Target audience:
- gym people
- students
- entrepreneurs
- gamers
- high performers

Brand emotions:
- focus
- ambition
- discipline
- energy
- momentum

Design direction:
- dark backgrounds
- magma/lava accents
- bold typography
- smooth animations
- premium UI
- fast loading
- modern ecommerce feel

Avoid:
- corporate supplement style
- generic Shopify look
- medical claims
- boring layouts

The experience should feel like:
“entering performance mode”.

Main slogan:
“Entrá en Modo Magma.”
